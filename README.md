# Day Greeting Loop

Цикл на Python от 0 до 23, который выводит текущее время и приветствие в зависимости от часа.

## Функциональность

- Цикл по всем часам суток (0-23)
- Вывод времени в формате "На часах X:00."
- Выбор приветствия по часу:
  - 0-5: Доброй ночи!
  - 6-11: Доброе утро!
  - 12-17: Добрый день!
  - 18-22: Добрый вечер!
  - 23: Доброй ночи!

## Пример работы

На часах 0:00.
Доброй ночи!
На часах 1:00.
Доброй ночи!
...
На часах 6:00.
Доброе утро!
...
На часах 23:00.
Доброй ночи!

## Куда можно встроить

- В программы с часами и приветствиями
- В виджеты времени
- В учебные материалы по циклам и условиям
- В демонстрацию работы elif

## Требования

- Python 3.x

## Примечание

Для часа 23 срабатывает else, так как условие current_hour < 23 не выполняется. Для часов 0-22 включительно срабатывают соответствующие ветки elif.
